## Node.js MVC style in this Next.js app

- `Model`: `app/modles/User.js` (Mongoose schema)
- `Service`: `app/services/auth-service.js` (business logic)
- `Controller`: `app/controllers/auth-controller.js` (request/response handling)
- `Router`: `app/api/**/route.js` (Next.js Route Handlers)

In Next.js App Router, `route.js` files act as routers/endpoints.
